# GenAI Things

A collection of experiments and projects exploring various aspects of Generative AI.

## About

This repository contains multiple experiments in the field of GenAI, including:

- **Tool Calling using Arcade**: Implementation of tool calling functionality using Arcade with open-source models (Ollama)
- More experiments coming soon...

## Getting Started

Each project folder contains its own implementation and dependencies. Navigate to the specific project directory for more details.

## Projects

### Tool Calling using Arcade
Demonstrates how to implement multiple tools using Arcade with open-source models like Ollama.

## Setup

1. Clone the repository
2. Install required dependencies for each project
3. Configure environment variables (create `.env` file based on project requirements)
