# GenAI Things

A collection of experiments and projects exploring various aspects of Generative AI.

## About

This repository serves as a workspace for multiple experiments in the field of GenAI, covering topics like tool calling, model integrations, prompt engineering, and more.

## Projects

### 1. Tool Calling using Arcade
Implementation of tool calling functionality using Arcade with open-source models (Ollama).

Location: `Tool Calling using Arcade/`

## Structure

Each project is organized in its own directory with:
- Jupyter notebooks or Python scripts
- Project-specific README (where applicable)
- Required configuration files

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/abishake-07/GenAI-Things.git
   ```

2. Navigate to a specific project directory

3. Install project-specific dependencies

4. Set up environment variables (create `.env` files as needed)

This is an experimental repository. Code and implementations are continuously evolving as new experiments are added.
